Marjor.am is a Game Management System

Marjor.am is a framework for building game servers. Inspired by content management systems (CMSs) like Wordpress and Drupal, Marjor.am aims to make setting up networking for your game much easier.

Marjor.am is Modular

While there is a default configuration we try to keep everything as modular as possible. We do this for two reasons. The first is for ease of development, as long as the internal interfaces stay the same we can work on each module as we see fit without disturbing the others.

The second is customization. We see Marjor.am as being a powerful framework for game developers, but we couldn't possibly anticipate every need of every game developer, so instead we made it as easy as possible to get up and running with as little custom code as possible. You may need to write a leaf to handle your "cyberpunk-wordgame-meets-baseball-sim" but you shouldn't need to touch any of Marjor.am's core.

Marjor.am is Light Weight

If you are looking to build the next World of EveCraft Online, this is probably not the framework for you. If you want to add profiles and a global high-score to your puzzle game, you are in the right place.

Marjor.am might be a good fit for your project if you need:

  • asynchronous communication (Marjor.am is not optimized for realtime)
  • low volume of data (Marjor.am is designed to run on basic webhosting)
  • simple data storage (think strings and numbers)

If that sounds like your project welcome to Marjor.am.
